Executável para iniciar instancias do Whaticket automaticamente.

Instale:

```bash
git clone https://github.com/henriquevpf/Start-instancia.git
```

Acesse o diretório:

```bash
cd Start-instancia
```

De permissões para o arquivo:

```bash
chmod +x start.sh
```

Execute:

```bash
./ Start-instancia.sh
```
